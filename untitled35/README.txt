S'assurer de la présence des fichiers : main.c, tree.c, tree.h, menu.c, menu.h, node.c, node.h, dictionnaire_non_accentue

Nous avons utilisé le dictionnaire non accentué.
Nous avons fait notre programme sur l'environnement Clion, tout autre environnement ne garantira pas le bon fonctionnement du code.

Lancer le programme, un menu s'affichera avec plusieurs options dans le terminal, il vous suffit de choisir l'option voulu en entrant le chiffre correspondant.
Il ne faut pas entrer une lettre à la place d'un chiffre.

Exemple : 
	
Si on veux rechercher une forme de base 
	
Affichage :	(1: RECHERCHER UNE FORME DE BASE
		...				)
	
On doit entrer 1 dans le terminal, puis entrer le mot voulu.
